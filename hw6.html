Part-1

This part is a tool where you enter a number with 4 decimal places or more. It calculates and shows the rounded number, square root, and more. If the number is invalid, it will display an error message.






Part-2

This part is a web tool that finds the desired character's number of appearance in the content provided. This tool is not capital letters or lowercase letters sensetive.





Part-3 (with extra credit)

This part is a web tool where the user enters the complete phone number then the program returns area code, first three digits and the last three digits.
